# Game
Здесь размещена попытка реализации части игры, а именно ход сражения. Имеются несколько существ Игрок и Монстр которые по очереди наносят урон друг другу при этом обновляются их жизни. \

У Игрока есть возможность восстановить себе 4 раза здоровье на 30% от максимального.

У существа нету спец. возможностей

Здовоье, атака, защита генерируются автоматически из диапазона 20-30
